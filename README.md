# Luxray

[**Discord**](https://discord.gg/9sfxrjpKdj) bot built to **support**, **monitor**, and **entertain** our **community**.
## 📦 What it does

- **🤖 Utility commands**
  Delivers useful information quickly and clearly.

- **📊 Community monitoring**
  Helps keep the server organized and active.

- **🎮 Fun interactions**
  Adds engaging features to improve the overall experience.

## 🎯 Purpose

Luxray is designed to:
- provide helpful tools ⚙️
- support and observe the community 👀
- make the server more enjoyable 🎉

All while staying **simple, reliable, and easy to use**.
